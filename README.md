## 🚀 部署方式（选择一种）

### 方式一：免费托管（推荐新手）

#### 1. GitHub Pages（免费）
1. 注册 GitHub 账号：https://github.com
2. 创建新仓库，命名为 `username.github.io`
3. 上传 `index.html` 文件
4. 访问 `https://username.github.io` 即可

#### 2. Vercel（免费+国内访问快）
1. 注册 Vercel：https://vercel.com
2. 点击 "Add New Project"
3. 上传文件或连接 GitHub
4. 获得免费域名：`xxx.vercel.app`

## 💳 支付配置

### 当前状态
支付弹窗已做好，但二维码需要手动配置。

### 配置步骤

1. **生成收款二维码**
   - 微信：打开微信 → 我 → 服务 → 收付款 → 扫码收款 → 保存二维码
   - 支付宝：打开支付宝 → 收钱 → 领取收钱码 → 保存二维码

2. **替换网站二维码**
   - 打开 `index.html`
   - 找到支付弹窗部分（约第480行）
   - 将收款码图片放到 `ai-tools-store` 文件夹
   - 修改 `<img>` 标签的 `src` 为你的二维码图片路径

```html
<div class="qr-placeholder">
    <img id="wechatQR" src="你的微信二维码.jpg" alt="微信支付二维码">
</div>
```
